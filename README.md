那么页面展示内容:
1.剩余可mint数量
2.选择mint 价值500U的/1000U的
3.选择mint30天的池子/mint60天的池子
4.动态展示mint后的每日收益和总收益
5.当前钱包还未mint：mint按钮
   当前钱包已经mint：领取收益按钮
